# AI PDF Teacher Assistant

Aplicación desarrollada en Python para ayudar a docentes a analizar documentos PDF mediante Inteligencia Artificial.

## Objetivo

Este proyecto permitirá:

- Cargar documentos PDF.
- Extraer el texto del documento.
- Generar resúmenes.
- Proponer actividades evaluativas.
- Generar preguntas automáticamente.
- Servir como base para un agente de IA orientado a docentes.

## Tecnologías (en construcción)

- Python
- Streamlit
- PyMuPDF
- FastAPI
- Git
- GitHub
